# Fearless and impeccable - Шаблоны для чатов

## Описание проекта

Это веб-приложение для управления шаблонами чатов, которое позволяет пользователям создавать, редактировать, удалять и организовывать шаблоны сообщений для различных ситуаций. По умолчанию приложение работает полностью на стороне клиента и использует IndexedDB или localStorage для хранения данных. Также доступен опциональный сервер для централизованного хранения.

## Функциональность

- Создание, редактирование и удаление шаблонов
- Организация шаблонов по категориям
- Поиск шаблонов по названию, категории или содержимому
- Фильтрация шаблонов по категориям
- Копирование шаблонов в буфер обмена
- Импорт и экспорт шаблонов в формате JSON или TXT
- Импорт шаблонов из текстовых (TXT) файлов
- Адаптивный дизайн для мобильных и десктопных устройств

## Технологии

- HTML5
- CSS3 (адаптивный дизайн)
- JavaScript (ES6+)
- IndexedDB / localStorage для хранения данных
- Express (опционально) для централизованного API
- Font Awesome для иконок

## Как запустить

### Вариант 1: Открыть файл напрямую

Просто откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Edge, Safari).

### Вариант 2: Использовать локальный сервер

Если у вас установлен Python:

```
python -m http.server 8000
```

Если у вас установлен Node.js:

```
npx http-server -p 8000
```

Затем откройте в браузере адрес: `http://localhost:8000`

## Использование

1. **Добавление шаблона**: Нажмите кнопку "Добавить шаблон", заполните форму и нажмите "Сохранить"
2. **Редактирование шаблона**: Нажмите на иконку редактирования в карточке шаблона
3. **Удаление шаблона**: Нажмите на иконку удаления в карточке шаблона
4. **Копирование шаблона**: Нажмите на иконку копирования в карточке шаблона
5. **Поиск шаблонов**: Используйте поле поиска в верхней части страницы
6. **Фильтрация по категориям**: Выберите категорию из выпадающего списка рядом с полем поиска
7. **Экспорт шаблонов**: Откройте "Экспорт" (через иконки на карточках или мультивыбор) и выберите формат JSON или TXT
8. **Импорт шаблонов**: Нажмите кнопку "Импорт", выберите вкладку JSON или TXT:
   - Для JSON: вставьте JSON и нажмите "Импортировать"
   - Для TXT: загрузите текстовый файл, укажите категорию по умолчанию и нажмите "Импортировать"

## Централизованное хранилище (опционально)

В проект добавлен простой сервер `server/` на Express для общего доступа к шаблонам.

Запуск API локально:

1. Перейдите в каталог `server/`:
   - `cd server`
2. Установите зависимости:
   - `npm install`
3. Запустите сервер:
   - `npm start`

Сервер поднимется на `http://localhost:3001` и будет хранить данные в `server/data/templates.json`.

Подключение фронтенда к серверу: в `index.html` перед подключением `js/app.js` добавьте:

```html
<script>
  window.REMOTE_API_URL = 'http://localhost:3001';
</script>
```

Когда указан `REMOTE_API_URL`, фронтенд будет использовать удалённый API вместо локального хранилища.

API эндпоинты:

- GET `/templates` — список
- POST `/templates` — создать `{ name, category, content, favorite? }`
- PUT `/templates/:id` — обновить
- DELETE `/templates/:id` — удалить
- POST `/templates/import-txt` — массовый импорт из TXT `{ txt, defaultCategory? }`

## Особенности реализации

- Приложение автоматически создает базу данных при первом запуске
- Если браузер не поддерживает IndexedDB, приложение автоматически переключается на использование localStorage
- При первом запуске создаются несколько шаблонов по умолчанию
- Все данные хранятся локально в браузере пользователя
